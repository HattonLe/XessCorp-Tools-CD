Making pin assignments for I/O using VHDL

Q:

I am using the XS40 board with VHDL. Can you tell me how to assign pin numbers for downloading in the VHDL source file, or any other method.

A:

You can use a .UCF file. If you have a signal called 'A' in your VHDL design that you want to assign to pin 16 of your FPGA, enter a constraint in the UCF file like so:

NET A LOC=P16;

If you have a bus declared like this: "B: out std_logic_vector(3 downto 0)" that you want to assign to pins 20-23, then use the following constraints:

NET B<0> LOC=P20;
NET B<1> LOC=P21;
NET B<2> LOC=P22;
NET B<3> LOC=P23;

Then instruct the Foundation software to look in the UCF file using the following menu items in the Project Manager window: Implementation=>Implementation Options. Type the name of your UCF file in the User Constraints field. Then run the implementation tools. You can check the pad report to make sure the pin assignment worked.
