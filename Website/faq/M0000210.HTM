Using old and new versions of XS Boards and XSTOOLs

Q:

With the NEW version of the XSTOOLs software the XS40 V1.3 board does not pass the test, but it does pass with the OLD version of the XSTOOLs. Can you tell me how to merge the old and new XSTOOLs so they work together so I can use either of my boards?

A:

To get a V1.3 (or older) XS40 Board and a V1.3 (or newer) XS95 Board to work with a single installation of XSTOOLs, you need to merge the OLD and NEW versions of the XSTOOLs software as follows:

  1. Uninstall all version of the XSTOOLs.
  2. Install the new version of the XSTOOLs in a directory called XSTOOLS-NEW.
  3. Install the old version of the XSTOOLs in a directory called XSTOOLS.
  4. Copy all files with a .SVF suffix from XSTOOLS-NEW\BIN to XSTOOLS\BIN.
  5. Erase (do not uninstall) the XSTOOLS-NEW directory.
  6. If you want, you can clean some of the extra XSTOOLs environment variables out of your AUTOEXEC.BAT after all this.

To get a V1.4 (or newer) XS40 Board and a V1.2 (or older) XS95 Board to work with a single installation of XSTOOLs, you need to merge the OLD and NEW versions of the XSTOOLs software as follows:

  1. Uninstall all version of the XSTOOLs.
  2. Install the new version of the XSTOOLs in a directory called XSTOOLS-NEW.
  3. Install the old version of the XSTOOLs in a directory called XSTOOLS.
  4. Copy all files with a .BIT suffix from XSTOOLS-NEW\BIN to XSTOOLS\BIN.
  5. Erase (do not uninstall) the XSTOOLS-NEW directory.
  6. If you want, you can clean some of the extra XSTOOLs environment variables out of your AUTOEXEC.BAT after all this.

If you have both old and new XS40 and/or XS95 Boards that you want to use on the same PC, then do the following:

  1. Uninstall all version of the XSTOOLs.
  2. Install the new version of the XSTOOLs in a directory called XSTOOLS-NEW.
  3. Install the old version of the XSTOOLs in a directory called XSTOOLS.
  4. When you want to use a V1.3 (or older) XS40 or V1.2 (or older) XS95, issue the following command in a DOS window: SET XSTOOLS_BIN_DIR=C:\XSTOOLS\BIN
  5. When you want to use a V1.4 (or newer) XS40 or a V1.3 (or newer) XS95, issue the following command in a DOS window: SET XSTOOLS_BIN_DIR=C:\XSTOOLS-NEW\BIN
  6. Then use the XSTOOLs utilities as normal. You do not need to reboot the PC after you set the XSTOOLS_BIN_DIR environment variable.
