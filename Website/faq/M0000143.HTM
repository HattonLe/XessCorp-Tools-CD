XESS support for Linux?

Q:

I think that XESS should support Linux systems to the same degree that you support win95 and NT systems.

A:

We have released the source code for the XSLOAD and XSPORT command line utilities. Go to http://www.xess.com/FPGA/ho07000.html to download it. There is some minimal documentation in the ZIP file and in the manuals/documentation section.

The source was compiled with Visual C++ 5.0.

This XStools source is in the public domain. So if you want a version that works with another compiler, please feel free to modify our source. If you want versions of XSLOAD and XSPORT that work under Linux, please feel free to create them. If you find errors, please feel free to fix them. If you see inefficiencies, please feel free to remove them. If the documentation is too sparse, please feel free to write more. We only ask that you make your results available to the rest of the community (either send the modified source code back to us or provide a link we can place on our site).

There is nothing we can do about the Foundation tools since those are obviously owned by Xilinx. They will port it to whatever OS they choose.
