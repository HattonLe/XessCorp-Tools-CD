What should my XS Board LED display when it is first turned on?

Q:

When I first turn on my XS 40 Board, the LED display shows '8' (all segments on). What happened with the board? Is this normal?

A:

The XS40 Board LED will normally show '8' when it is first turned on because the outputs have weak pull-up resistors that will dimmly activate the LEDs.

The XS95 Board LED will normally be blank when it is first turned on. However, some of the LED segments may be on if you have programmed the CPLD with a design that drives the LED.

The XSA Board LED will normally display a '7' when it is first turned on.

The XSV Board LEDs will normally be blank when it is first turned on.

The XSB Board LEDs will normally be blank when it is first turned on.
