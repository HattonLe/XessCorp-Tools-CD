JTAGPort ( Class )
Low-level and high-level JTAG operations.
 
Description
Low-level and high-level JTAG operations.

This object provides JTAG capabilities. It stores the state of the JTAG TAP state machine and updates the state as the TMS and TCK signals change state. It provides methods that make it easier to move between states of the TAP machine and to access the boundary scan instruction and data registers.

This object will be inherited along with either the parallel port or USB port object to provide JTAG capabilities to an actual physical port.
