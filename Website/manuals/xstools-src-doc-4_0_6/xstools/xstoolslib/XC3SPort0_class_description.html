XC3SPort ( Class )
Download configuration bitstreams into a Spartan3 FPGA.
 
Description
Download configuration bitstreams into a Spartan3 FPGA.

This object provides methods for downloading a configuration bitstream into a Spartan3 FPGA. The bitstream file is opened and the field containing the configuration bits is extracted and passed byte-by-byte into the FPGA. The download operates in one of two modes: fast mode where each byte is downloaded through the parallel port and into the Spartan3 FPGA in SelectMap configuration mode, or slow mode where each byte is serially transmitted through a single pin of the parallel port into the FPGA in slave-serial configuration mode.
