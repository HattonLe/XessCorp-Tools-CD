Description
Uploading and downloading of hexadecimal records to the Spansion Flash memory on XSA-200 and XSA-3S1000 Boards.

This object specializes the FlashPort object by providing methods for erasing blocks and writing individual bytes of a Spansion S29* Flash memory.
