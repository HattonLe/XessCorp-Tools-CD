Description
Load the BSIR with an instruction, execute the instruction, and then capture and reload the BSDR. This subroutine assumes the TAP controller is in the Test-Logic-Reset or Run-Test/Idle state upon entry. The TAP controller is returned to the Run-Test/Idle state upon exit.
